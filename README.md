Backbone-Webpack Example
===============

[![Build Status][travis-image]][travis-url]

1）checkout 分支代码

2）安装依赖

    $ cd webapp
    $ npm install

3）进入开发环境

    npm run d

访问：http://localhost:3000，Ctrl + C 退出

4）构建生成环境代码

    npm run p

**biz-dp 开发中，暂时不能在编译平台构建，请在提交代码之前构建！**
